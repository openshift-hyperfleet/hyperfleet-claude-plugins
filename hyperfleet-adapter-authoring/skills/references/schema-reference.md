# Adapter Configuration Schema Reference

## How to Use

Fetch the adapter configuration schema and documentation from the adapter repo:

```text
https://raw.githubusercontent.com/openshift-hyperfleet/hyperfleet-adapter/main/docs/adapter-authoring-guide.md
```

For architecture-level standards (e.g., adapter status contract, condition types), use the `hyperfleet-architecture` skill.

Do NOT hardcode schema content here -- always fetch the latest from the source repos.
