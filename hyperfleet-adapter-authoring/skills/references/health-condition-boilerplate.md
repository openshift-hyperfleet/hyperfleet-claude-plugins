# Health Condition Boilerplate

## How to Use

Read the adapter authoring guide:

```bash
curl -sL https://raw.githubusercontent.com/openshift-hyperfleet/hyperfleet-adapter/main/docs/adapter-authoring-guide.md
```

Or browse it directly on GitHub: `https://github.com/openshift-hyperfleet/hyperfleet-adapter/blob/main/docs/adapter-authoring-guide.md`

Look for the "Health condition" section in the authoring guide. Copy the standard Health condition YAML exactly as documented -- do not modify it.

For the adapter status contract and condition semantics, use the `hyperfleet-architecture` skill to fetch the relevant architecture document.
